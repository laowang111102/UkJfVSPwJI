# 项目简介 java1081

本项目是一款基于Vue + SpringBoot技术的课程管理系统。该系统实现了前后端分离，界面美观，使用当前流行的技术栈，确保系统的高效稳定。

## 技术栈

### 前端

- Vue.js
- ElementUI
- axios

### 后端

- SpringBoot
- 持久层：Mybatis Plus
- 授权认证：手写实现
- 会话管理：Spring Session + Redis
- 日志记录：AOP + MongoDB

### 数据库

- MySQL（MariaDB）：存储核心业务数据
- Redis：存储Session、新闻、配置信息
- MongoDB：存储业务日志

## 功能模块

- 课程管理
- 选课系统
- 用户权限管理
- 日志记录

## 系统角色

- 学生
- 教师
- 管理员

## 运行环境

- 前端：现代浏览器
- 后端：Java 8 或更高版本

## 部署步骤

1. 部署后端服务
   - 部署SpringBoot应用
   - 配置数据库MySQL、Redis和MongoDB
2. 部署前端服务
   - 构建Vue项目
   - 部署至Web服务器

## 目录结构

```
项目名称/
├── frontend/   # 前端项目目录
└── backend/    # 后端项目目录
    ├── src/
    ├── resources/

```

## 核心亮点

- 前后端分离，便于维护和扩展
- 采用分布式Session管理，提高系统性能
- 利用AOP技术实现日志记录，便于追踪和监控
- 支持多种数据库，适应不同业务场景需求
